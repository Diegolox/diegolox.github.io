## Proyectos destacado

### [TheBUG01](https://github.com/Diegolox/TheBUG01)

**Robot diferencial plug-and-play para prototipado rápido y educación**

TheBUG01 es una plataforma robótica basada en una **PCB todo-en-uno** para un robot diferencial de dos ruedas, pensada para reducir cableado, acelerar el prototipado y facilitar la integración de hardware y software.

**Aspectos técnicos principales**
- PCB de control con periféricos integrados
- Basado en **ESP32**
- **IMU** para estimación de orientación y movimiento
- **3 sensores TOF400 por I²C** para detección de obstáculos
- Driver dual **TB6612FNG** para control de motores DC
- Convertidor **DC-DC** integrado
- Protección frente a **back-EMF / regeneración**
- Diseño orientado a educación y robótica experimental
