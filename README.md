## Proyectos destacados

### 1) [Nombre del proyecto 1](https://github.com/TU-USUARIO/REPO-1)

**Control de velocidad de motor DC con PID**

Proyecto orientado al diseño e implementación de un lazo de control para regular la velocidad de un motor DC, incluyendo adquisición de señales, ajuste del controlador y validación experimental.

**Qué hice:**

* Modelado básico del sistema
* Implementación del controlador
* Adquisición y procesamiento de señales
* Validación con métricas de desempeño

**Tecnologías:** `C/C++` `MATLAB/Simulink` `PID` `PWM`

**Resultado destacado:** error en régimen < 2%, tiempo de establecimiento de 0,8 s

### 2) [Nombre del proyecto 2](https://github.com/TU-USUARIO/REPO-2)

**Sistema embebido de monitorización y telemetría**

Desarrollo de un nodo embebido para lectura de sensores, procesamiento local y envío de datos a una interfaz de supervisión.

**Qué hice:**

* Integración de sensores
* Gestión de comunicaciones
* Filtrado y tratamiento de señales
* Envío de datos y visualización

**Tecnologías:** `STM32` `Python` `MQTT` `I2C` `SPI`

**Resultado destacado:** latencia media de 35 ms y alta estabilidad en pruebas
